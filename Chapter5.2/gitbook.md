# gitbook

# 安装

1.  安装node.js和npm

```bash
 # Mac OSX.
 $ brew install node npm

 # Ubuntu.
 $ sudo apt-get install nodejs npm
```

2.  安装n

```bash
$ sudo npm install n -g
```

3. 安装gitbook-cli

```bash
 $ sudo npm install -g gitbook-cli
```

4. 检查版本

```bash
 $ gitbook -V
 CLI version: 2.3.0
 GitBook version: 3.1.1
```

> 参考: https://github.com/chusiang/how-to-build-the-gitbook-with-gitbook-cli
